# Reinforcement Graph Overlapping Clustering (RGOC)

Main execution script for training the Reinforcement Graph Overlapping Clustering (RGOC) model. The core concept of this framework is built on the “Reinforcement Graph Clustering with Unknown Cluster Number” (RGC) architecture proposed by Liu et al. (2023).

### Quick Start

- Step 1: Ensure you have the **facebook** dataset unzipped into the **./dataset** folder.
- Step 2: Run the training script. By default, it will train and evaluate on the facebook ego networks:

```bash
python train.py
```
